<article>

Применяется для статей сайта, сообщений блога и форума, комментариев.

          
<article>
  <h1>Заголовок</h1>
  <p>Текст статьи</p>
</article>

Элемент <article> обязательно должен содержать <h1>, <h2> <h3> <h4> <h5> или <h6>
