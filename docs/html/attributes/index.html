HTML атрибуты

DOM-объекты и их свойства генерируется на основании html-тегов и их атрибутов.

Например для тега <section id="news"> будет создано свойство section.id="news"

          
<section id="news">

Метод hasAttribute

Проверяет налчие атрибута у элемента (тега), где в скобках указывается имя проверяемого атрибута.

          
          let elem = document.querySelector('#news');
console.log(elem.hasAttribute('id'));

Метод getAttribute

Проверяет значение атрибута, где в скобках так-же указывается имя проверяемого атрибута.

          
          let elem = document.querySelector('#news');
console.log(elem.getAttribute('id'));

Стандартные и нестандартные атрибуты

У html-тегов есть список стандартных для него атрибутов.

Например, для элемента <section> доступны доступны глобальные атрибуты:

accesskey class contenteditable dir draggable hidden id lang spellcheck style tabindex title translate

Эксперимент

Создам элемент <section> и пропишу в нем все глобальные атрибуты (которые являются стандартными) и один нестандартный атрибут.

          
<section accesskey="ok" class="ok" contenteditable="ok" dir="ok" draggable="ok" hidden="ok" id="ok" lang="ok" spellcheck="ok" style="ok" tabindex="ok" title="ok" translate="ok" non-standart-attr="ok">

        
          
          let elem = document.querySelector('#ok');

console.log(elem.hasAttribute('accesskey'));
console.log(elem.hasAttribute('class'));
console.log(elem.hasAttribute('contenteditable'));
console.log(elem.hasAttribute('dir'));
console.log(elem.hasAttribute('draggable'));
console.log(elem.hasAttribute('hidden'));
console.log(elem.hasAttribute('id'));
console.log(elem.hasAttribute('lang'));
console.log(elem.hasAttribute('spellcheck'));
console.log(elem.hasAttribute('style'));
console.log(elem.hasAttribute('tabindex'));
console.log(elem.hasAttribute('title'));
console.log(elem.hasAttribute('translate'));
console.log(elem.hasAttribute('non-standart-attr')); // false
