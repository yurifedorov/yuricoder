Основы HTML

Hypertext Markup Language (HTML) - не является языком программирования, он предназначин для разметки и дает понять браузеру как располагать элементы на странице.

Исчерпывающее руководство: HTML 5.2 W3C Recommendation

HTML элемент

Пример элемента <h1>, для обозначения заголовка верхнего уровня:

          
<h1>Заголовок</h1>
  • открывающий тег <h1>
  • закрывающий тег </h1>
  • любой контент (текст)

Все вместе это называется Element

Атрибуты HTML элемента

Содержат дополнительную информацию, которая не видна при отображении страницы в браузере

          
<h1 class="title">Заголовок</h1>
  • class - имя атрибута
  • title - значение атрибута

Теперь элемент <h1> имеет метку 'title', что позволит обратиться к нему в CSS и задать стилевое офермление.

Атрибутов много и для каждого тега есть список стандартных атрибутов.

Например исчерпывающая информация о элементе <h1> есть в официальной спецификации

Вложенность элементов

Все элементы, кроме <html> могут быть вложенными.

          
<!DOCTYPE html>
<html>
  <head>
    <title>Заголовок</title>
  </head>
  <body>
    <h1>Заголовок</h1>
    <p>Lorem ipsum dolor...</p>
  </body>
</html>

Вложенность отчетлива видна в DOM деревее, где <html> (родитель для всех), <head> и <body> - его прямые потомки, у которых тоже есть потомки <title>, <h1> и <p>.

Пустые элементы

Это элементы, которые не имеют контента.

          
<img src="picture.jpg" alt="description">

Элемент <img> не имеет контанта и закрывающего тега, он предназначен для вставки изображения

Базовая разметка

Основа любой html страницы, так сказать самый минимум.

          
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
  </head>
  <body>
    <!-- контент -->
  </body>
</html>
<!DOCTYPE html> просто должен быть
<html> корневой элемент
<head> контейнер для подключаемых стилей, мета-данных и много чего еще
<body> содержит видимую часть html страницы: заголовки, абзацы, картинки, таблицы, списки и т.д.
<meta> информация для браузера или поисковых систем
<title> заголовок страницы, который отображается на вкладке браузера и в поисковиках

Заголовки

Всего в HTML 6 уровней заголовков (хотя обычно используется первые 3).

  1. <h1> - главный заголовок (может быть только один на странице)
  2. <h2> - подзаголовок
  3. <h3> - под-подзаголовок
  4. <h4> - под-под-подзаголовок
  5. <h5> - под-под-под-подзаголовок
  6. <h6> - под-под-под-под-подзаголовок

Важно всегда соблюдать иерархию заголовков!

Элемент <h1> является родителем для всех остальных заголовков.

Внутри секции с заголовком <h3> нельзя задать подзаголовки с <h2>

Ниже пример семантически неверной расстановки заголовков:

          
<h1>Модели автомобилей</h1>

<section>
  <h3>Внедорожники</h3>
  <p>Ниже представлен список внедорожников</p>

  <h2>Toyota Land Cruiser 200</h2>
  <p>крутой внедорожник</p>

  <h2>Suzuki Grand Vitara</h2>
  <p>немножко подешевле</p>
</section>

Фавиконка (favicon)

Миниатюрный значек который находится на вкладке браузера рядом с названием страницы.

          
<link rel="icon" href="favicon.ico">

Бывают анимированные favicon - то .gif файлы, содержащие анимацию
