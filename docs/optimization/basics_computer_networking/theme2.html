Протоколы верхнего уровня

Фактически сейчас используется упрощенная сетевая модель TCP/IP, которая в отличии от OSI состоит из 4-х уровней: application, transport, internet и link or network interface.

  • Application Layer - компетенция програмистов
  • Data Flow Layer - компетенция системных администраторов
