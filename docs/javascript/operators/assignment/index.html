Присваивание производится знаком равно ( = ).

          
          var x = "value";
alert( x );

Приоритет присваивания

Оператор присваивания имеет низкий приоритет, поэтому сначала всегда производится арифметика, а затем результат присваивается переменной.

          
          var x = 2 * 2 + 1;
alert( x ); // 5
5 сначала выполнены арифметические операции, затем результат присваивается переменной x

Направление присваивания

Производится справа налево.

          
          var a, b, c;
a = b = c = 2 + 2;
alert( a ); // 4
alert( b ); // 4
alert( c ); // 4
4 сначала вычислится самое правое выражение 2+2 и присвоится переменной c
4 затем c присвоится b (b = c)
4 затем b присвоится a (b = a)

Сокращенный синтаксис

При помощи совмещённых операторов можно укорачивать запись

          
          // обычный способ
var n = 2;
n = n + 5;
n = n * 2;
alert(n); // 14

// короткий способ
var n = 2;
n += 5;
n *= 2;
alert(n); // 14
14 сначала n + 5 (2 + 5), потом и n * 2 (7 * 2)
14 n=7 (работает как n = n + 5), n=14 (работает как n = n * 2)
