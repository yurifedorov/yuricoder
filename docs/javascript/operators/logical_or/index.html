Если хотя бы один аргумент true, то возвращается true, иначе false .

          
          alert( true || true ) // true
alert( false || true ) // true
alert( false || false ) // false

Если один из аргументов true, то выводится сообщение 'success'.

          
          var a = 1, b = '0';

if (a && b) {
  alert('success');
}
b = '0' Если значение аргумента не логического типа (true или false), то в целях вычисления оно к нему приводится автоматически.

Число 1 воспринимается как true, число 0 false.
