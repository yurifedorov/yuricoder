Object Объекты Deep Object Mutability

object

          
          let person = {   // объект "person"
  number: 1,     // ключ "number", значение 1
  name: "Mike",  // ключ "name", значение "Mike"
  age: 25        // ключ "age",  значение 25
};

/* обращение к свойствам объекта */
alert(person.number); // 1
alert(person.name);   // Mike
alert(person.age);    // 25

Deep Object Mutability

          
          const user = {
  name: 'Joe',
  age: 25,
  pet: {
    type: 'dog',
    name: 'Buttercup'
  }
};

Object.freeze(user);

user.pet.name = 'Daffodil';

console.log(user.pet.name); // Daffodil
JavaScript knowledge quizzes by Nick Scialli

✗ Объяснение

Object.freeze выполнит поверхностное замораживание объекта, но не защитит глубокие свойства от изменения. В этом примере мы не сможем мутировать user.age, но у нас нет проблем с мутированием user.pet.name. Если мы чувствуем, что нам нужно защитить объект от мутации "полностью вниз", мы можем рекурсивно применить Object.freeze или использовать существующую библиотеку "deep freeze".
