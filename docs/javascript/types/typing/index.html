Типизация в программировании

Типизация бывает статическая или динамическая, их отличия в следующем:

статическая динамическая 
          
          // int var1 = 10;
  • переменная связывается с типом в момент объявления
  • переменные привязаны к типам
  • тип не может быть изменён позже

Пример языка: Java

 
          
          let var1 = 10; // тип number
  • переменная связывается с типом в момент присваивания
  • переменные не привязаны к типам
  • в различных участках программы одна и та же переменная может принимать значения разных типов
