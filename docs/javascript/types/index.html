Типы в JavaScript

JavaScript является динамически типизированным языком. Одна и таже переменная может иметь значение разного типа.

          
          let x = 10;  // string
x = 10;      // number

Сначала значение переменной x было string, потом стало number.

Основные типы в JS

  1. number для чисел
  2. string для строк
  3. boolean для true/false
  4. null для неизвестных значений
  5. undefined для неприсвоенных значений
  6. object для более сложных структур данных
  7. symbol для уникальных идентификаторов

Оператор typeof

Оператор typeof возвращает тип аргумента

          
          alert(typeof undefined); // "undefined"
alert(typeof 0); // "number"
alert(typeof true); // "boolean"
alert(typeof "foo"); // "string"
alert(typeof Symbol("id")); // "symbol"
alert(typeof Math); // "object"  (1)
alert(typeof null); // "object"  (2)
alert(typeof alert); // "function"  (3)
