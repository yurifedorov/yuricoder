Объявление функций

Здесь я попытаюсь понять какие бывают способы объявления функций, когда лучше использовать тот или иной способ, а так-же прочие тонкости.

Способы объявления

  • method #1

    объявление функции 
              
          function A() {
  /* тело функции */
};

  • method #2

    функциональное выражение 
              
          var B = function() {
  /* тело функции */
};

  • method #3

    функциональное выражение с группирующими операторами 
              
          var C = (function() {
  /* тело функции */
});

  • method #4

    именованное функциональное выражение 
              
          var D = function foo() {
  /* тело функции */
};

  • method #5

    немедленно исполняемое функциональное выражение (IIFE), которое возвращает функцию 
              
          var E = (function(){
  return function() {
    /* тело функции */
  }
})();

  • method #6

    конструктор Function 
              
          var F = new Function();

  • method #7

    особый случай: конструктор объекта 
              
          var G = new function() {
  /* тело функции */
};

Function definition

через ключевое слово 'function'

          
          function funcName(arg) { // parameter (arg)
  alert('Phone: ' + arg);
}

funcName('+7-111-11-11'); // Phone: +7-111-11-11

При вызове функции funcName(), значения в скобках копируются в локальную переменную arg, затем используются в теле функции.

Function expressions

в виде выражения

          
          let funcName = function(arg) { return 'Phone: ' + arg; };
let phone = funcName('+7-111-11-11');

alert(phone); // 'Phone: +7-111-11-11
