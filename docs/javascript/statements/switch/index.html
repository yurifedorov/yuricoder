Конструкция "switch"

Конструкция switch

Синтаксис

Конструкция включает в себя один или несколько блоков case и необезательный блок default

          
          let str = prompt('2 + 2 = ?', '');
let num = Number(str);

switch (num) {
  case 3:
    alert( 'почти угадал' );
    break;
  case 4:
    alert( 'верно' );
    break;
  case 5:
    alert( 'почти угадал' );
    break;
  default:
    alert( "не верно" );
}

Что здесь происходит?

1 метод prompt() возвращает string
2 Number(str) преобразует string в number и сохряняет в переменной num
3 переменная num проверяется на строгое равенство каждому case
4 оператор break прерывает выполнение текущего цикла

Не забывай про break!

Если break не задан, то инструкции будут выполнятся по очереди до конца switch

          
          let str = prompt('2 + 2 = ?', '');
let num = Number(str);

switch (num) {
  case 3:
    alert( 'почти угадал' );
    //break;
  case 4:
    alert( 'верно' );
    //break;
  case 5:
    alert( 'почти угадал' );
    //break;
  default:
    alert( "не верно" );
}

Что произойдет теперь?

Если ввести '3' то выполнение начнется с 'case 3' и до конца, при этом все остальные проверки проигнорируются.

Групировка case

Блоки case с одинаковыми инструкциями можно объеденить в группы, что позволяет сделать код немного короче и гармоничнее

          
          let str = prompt('2 + 2 = ?', '');
let num = Number(str);

switch (num) {
  case 3:
  case 5:
    alert( 'почти угадал' );
    break;
  case 4:
    alert( 'верно' );
    break;
  default:
    alert( "не верно" );
}

По факту такая возможность это последствие того, как switch работает без break.

Проверяй тип значения!

В switch/case проверка на равенство всегда строгая, а значит должен соответствовать и тип значения

          
          let str = prompt('2 + 2 = ?', '');
//let num = Number(str);

switch (num) {
  case 3:
    alert( 'почти угадал' );
    break;
  case 4:
    alert( 'верно' );
    break;
  case 5:
    alert( 'почти угадал' );
    break;
  default:
    alert( "не верно" );
}

Что здесь не так?

В текущих блоках case выполняется сверка чисел 3, 4 и 5, а метод prompt() возвращает строки '3', '4', '5' и тд - поэтому ничего не работает.

Чтобы решить проблему нужно просто преобразовать string в number с помощью Number(str)
