Несколько условий: «else if»

Конструкция else...if

Код, находящийся в блоке if, выполняется если условие является true, если условие false, выполняется блок else.

Примеры

условие == false

          
          if (1==2) {
  alert('условие');
} else {
  alert('else');
}

Выполняется инструкция else (условие false)

условие == true

          
          if (2==2) {
  alert('условие');
} else {
  alert('else');
}

Выполняется инструкция if (условие true)

блок else отсутствует

          
          if (1==2) {
  alert('условие');
}

Ничего не произойдет (условие false и else не задан)

несколько условий, где первое true

          
          if (2 == 2) {
  alert('условие1'); // выполняется
} else if (4 == 4) {
  alert('условие2'); // игнорируется
} else {
  alert('else'); // игнорируется
}

Выполняется первая инструкция if, остальное игнорируется

несколько условий, где второе true

          
          if (1 == 2) {
  alert('условие1'); // false
} else if (4 == 4) {
  alert('условие2'); // выполняется
} else {
  alert('else'); // игнорируется
}

Выполняется вторая инструкция if, остальное игнорируется

несколько условий, где оба false

          
          if (1 == 2) {
  alert('условие1'); // false
} else if (2 == 4) {
  alert('условие2'); // false
} else {
  alert('else'); // выполняется
}

Оба условия false, выполняется else

Использование оператора ?

Оператор 'вопросительный знак' (или 'тернарный оператор') позволяет делать более короткую запись, когда нужно присвоить значение переменной в зависимости от условия.

Два примера ниже выполняют одно и тоже, только во втором конструкция if...else заменена оператором ?

длинная запись

          
          // let result;
let au_capital = prompt('Столица Австралии?', '');
if (au_capital == 'Канберра') {
  result = true;
} else {
  result = false;
}
alert(result);

P.S. убрал объявление переменной result, думал будет ошибка вроде 'переменная не объявлена'

Но нет, все работает! Как это так это??

короткая запись

          
          let au_capital = prompt('Столица Австралии?', '');
let result = (au_capital == 'Канберра') ? true : false;
alert(result);

Сначала вычисляется условие: если оно истинно, тогда возвращается true, если ложно – false.
