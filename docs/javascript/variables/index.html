Переменные в JavaScript

Переменные нужны для хранения информации, с которой в дальнейшем будет работать JS-приложение.

Для объявления переменной используется ключевое слово let.

          
          let storage;
storage = 'какие-то данные';
alert(storage);
1. объявлена переменная с именем 'storage'
2. с помощью оператора = в нее помещены данные
3. с помощью alert() переменная выводится на экран
Существует еще устаревшее ключевое слово var, работает оно по-другому.

Список зарезервированных слов

Эти слова нельзя использовать в качестве имён переменных, потому что они используются самим языком.
