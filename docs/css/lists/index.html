.list

Для незначительных изменений (размер шрифта, тип маркеров и т.д.)

          
<ul class="list">
  <li>item-1
  <li>item-2
  <li>item-3
  <li>item-4
  <li>item-5
  <li>item-6
  <li>item-7
  <li>item-8
</ul>

        
          .list {
  font-size: 1.2em;
}

.list > li {
  list-style-type: square;
}
