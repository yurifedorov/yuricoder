.tile_flip (модификатор)

Модификатор flip добавляет возможность переворачивать элементы при наведении мышы.

Добавляя модификатор _flip, все элементы содержащие контейнер .back станут переворачиватся при наведении курсора (в примере ниже - это третий элемент).

          
<ul class="tile tile_flip">
  
  <li><a href="#">.com</a></li>

  <li>
    <div>
      <a href="#">.dev</a>
      <p>некий текст</p>
    </div>
  </li>

  <li>
    <div class="front">
      <a href="#">.org</a>
      <p>просто текст</p>
    </div>
    <div class="back">
      <p>back side qqqqqqqqqqqq</p>
      <button>кнопка</button>
    </div>
  </li>

  <li>
    <div class="front">
      <a href="#">.biz</a>
      <p>просто текст</p>
    </div>
    <div class="back">
      <p>back side qqqqqqqqqqqq</p>
      <button>кнопка</button>
    </div>
  </li>

  <li><a href="#">.biz</a></li>
  <li><a href="#">.info</a></li>
  <li><a href="#">.net</a></li>
  <li><a href="#">.ru</a></li>
  <li><a href="#">.au</a></li>

</ul>

        
          .tile_flip div {
  transition: all .5s;
  backface-visibility: hidden; /* скрывает задний текст */
}

/* контейнер .back */
.tile_flip .back {
  display: grid;
  transform: rotateY(-180deg);
}

.tile_flip li:hover > .front {
  transform: rotateY(180deg);
}

.tile_flip li:hover > .back {
  transform: rotateX(0deg);
  background: var(--backside-color);
}
